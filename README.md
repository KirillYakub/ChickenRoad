<h3>Запуск:</h3>
<p>Запуск можливий через завантаження прикріпленого apk-release файлу або через git clone.</p>
<h3>Логіка гри:</h3>
<ul>
  <li>Кожен рівень містить 2, 4 чи 6 елементів, які треба знайти.</li>
  <li>При натисканні на правильний елемент картка не закриється, при натисканні на неправильний елемент буде видно деякий час і після він пропаде.</li>
  <li>Перемога на рівні зараховується після знаходження всіх елементів, програш після закінчення часу.</li>
  <li>Час на рівні зупиняється при згортанні програми або переході на екран налаштувань з екрану гри.</li>
</ul>
<h4>Інше:</h4>
<ul>
  <li>Щоб побачити себе в рейтинговому списку потрібно пройти хоча б один рівень.</li>
</ul>
